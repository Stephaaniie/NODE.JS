# Notas:
Opciones para poder hacer uso del programa
```
$ node app --help
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Es la base de la tabala de multiplicar      [number] [required]
  -h, --hasta    Es el limite que se desea que se calcule la tabla de
                 multiplicar                              [number] [default: 10]
  -l, --listar   Muestra la tabla por consola
                                           [boolean] [required] [default: false]

```
